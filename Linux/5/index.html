Voorbeeld

Stel dat deze oefening initieel de volgende directorystructuur heeft:

copy
  |
  +- source
       |
       +-- a.txt
       +-- b.txt
       +-- c.txt
       +-- d.txt
       +-- e.txt

Na het uitvoeren van je commando’s moet de nieuwe toestand zijn:

copy
  |
  +- source
  |    |
  |    +-- a.txt
  |    +-- b.txt
  |    +-- c.txt
  |    +-- d.txt
  |    +-- e.txt
  |
  +- target
       |
       +-- a.txt
       +-- b.txt
       +-- c.txt
       +-- d.txt
       +-- e.txt
