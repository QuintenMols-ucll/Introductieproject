Github Authentication

Table of Contents
  • Opdracht

Als je git gebruikt om informatie van GitHub te halen moet je je authenticeren bij GitHub.

Dit kan op 3 manieren:

  1. Via username/password, net zoals je je aanmeldt op de website

  2. Via een token dat dienst doet als (tijdelijk) wachtwoord

  3. Via SSH keys

Suport voor username/password is in de loop van 2021 afgesloten omdat deze manier niet veilig is.

Meer informatie over aanmelden met een token kan je hier vinden. Dit wordt veel gebruikt in bijvoorbeeld code editors met ingebouwde git functionaliteit (IntelliJ bijvoorbeeld). Je kan dan een token aanmaken specifiek voor deze applicatie.

De meest veilige manier om aan te melden is via SSH. Als je wil weten wat SSH is en waarom het erg veilig is kan je hier achtergrond informatie vinden.
